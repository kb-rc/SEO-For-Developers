sitemap.xml

An XML sitemap is a structured file that lists all crawlable URLs on your site along with metadata such as last modification date. It is submitted to search engines via Google Search Console and referenced in robots.txt to help crawlers discover content efficiently.

Standard sitemap

The following is a subset of the actual sitemap.xml deployed for this site, showing the first three entries. The full sitemap covers all 17 pages in the documentation.

sitemap.xml
XML 
<?xml version="1.0" encoding="UTF-8"?>
<urlset
  xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"
  xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
  xsi:schemaLocation="http://www.sitemaps.org/schemas/sitemap/0.9
    http://www.sitemaps.org/schemas/sitemap/0.9/sitemap.xsd">

  <url>
    <loc>https://kb-rc.github.io/SEO-For-Developers/</loc>
    <lastmod>2026-03-26</lastmod>
    <changefreq>weekly</changefreq>
    <priority>1.0</priority>
  </url>

  <url>
    <loc>https://kb-rc.github.io/SEO-For-Developers/docs/title-tags.html</loc>
    <lastmod>2026-03-26</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>

  <url>
    <loc>https://kb-rc.github.io/SEO-For-Developers/docs/meta-descriptions.html</loc>
    <lastmod>2026-03-26</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>

  <!-- ... remaining 14 URLs omitted for brevity -->

</urlset>

Sitemap index for large sites

Sites with more than 50,000 URLs, or sitemaps exceeding 50MB uncompressed, must split their content into multiple sitemap files coordinated by a sitemap index. The index file itself does not contain <url> entries - it only references child sitemaps.

sitemap-index.xml
XML 
<?xml version="1.0" encoding="UTF-8"?>
<sitemapindex xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">

  <sitemap>
    <loc>https://example.com/sitemap-pages.xml</loc>
    <lastmod>2026-03-26</lastmod>
  </sitemap>

  <sitemap>
    <loc>https://example.com/sitemap-posts.xml</loc>
    <lastmod>2026-03-26</lastmod>
  </sitemap>

  <sitemap>
    <loc>https://example.com/sitemap-products.xml</loc>
    <lastmod>2026-03-26</lastmod>
  </sitemap>

</sitemapindex>
Google largely ignores the <priority> and <changefreq> fields in sitemaps. Focus on providing accurate <lastmod> dates - Google uses these to determine recrawl priority. Only include URLs that return HTTP 200 and do not carry a noindex directive. Submitting noindex or redirect URLs wastes crawl budget.
