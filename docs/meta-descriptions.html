Meta Descriptions

The description meta tag provides a summary shown in SERPs below the title. It does not directly influence ranking, but affects click-through rate (CTR) - a secondary ranking signal.

Rules

  • Length: 120–158 characters. Google truncates descriptions at approximately 920px display width, which corresponds to roughly 158 characters on desktop and fewer on mobile. Aim for 120–158 characters to maximize visible content without truncation.
  • Action-oriented language. Begin with a verb or value proposition: "Learn how to…", "Discover the exact…", "A complete reference for…". This increases click intent by making the benefit immediately clear.
  • Include the primary keyword naturally. Google bolds terms in the snippet that match the user's search query. Naturally including the keyword increases visual prominence in SERPs.
  • Unique per page. Duplicate meta descriptions across multiple pages are flagged in Google Search Console and deprive each page of its opportunity to attract the right audience segment.
index.html
HTML 
<meta name="description"
      content="A complete technical SEO reference for developers. Covers title tags, meta descriptions, structured data, sitemaps, robots.txt, canonical URLs, Core Web Vitals, and more - with real file examples." />

<meta name="description"

Google rewrites descriptions approximately 62% of the time. Even so, writing a strong description sets the baseline and controls the narrative when Google does not rewrite it. It is also used verbatim by other search engines and social scrapers that do not rewrite snippets.
