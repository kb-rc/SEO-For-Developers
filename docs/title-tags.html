Title Tags

The <title> element defines the page title shown in browser tabs and as the clickable headline in search results. It is the single highest-impact on-page SEO element.

Rules

  • Length: 50–60 characters. Google truncates titles at approximately 580px display width (roughly 60 characters). Shorter titles leave ranking potential unused; longer titles are cut in SERPs.
  • Primary keyword first. Place the most important keyword or phrase as close to the start of the title as possible. Search engines weight earlier terms more heavily, and users reading SERPs scan left-to-right.
  • Brand suffix last. Append the brand name at the end separated by an em-dash (-) or pipe (|). Example: Title Tags - SEO Reference. Placing the brand first wastes the most valuable characters on a term that rarely matches search queries.
  • Unique per page. Every page on the site must have a distinct title. Duplicate titles prevent search engines from differentiating pages and dilute ranking signals.
  • No keyword stuffing. Repeating keywords or cramming synonyms degrades readability and can trigger spam filters. One primary keyword and one optional modifier are sufficient.

Format

Below are the exact title tags used in the two primary files of this site.

index.html
HTML 
<title>Technical SEO Reference for Developers &mdash; SEO Reference</title>
docs/title-tags.html
HTML 
<title>Title Tags &mdash; SEO Reference</title>

Dynamic titles via JS

For single-page applications where server-side rendering is not available, document.title can be set client-side. This is a fallback - prefer a static <title> in HTML whenever possible, as Googlebot may not execute JavaScript during initial indexing.

The setPageTitle utility in assets/js/seo.js enforces the keyword-first, brand-last convention:

assets/js/seo.js
JS 
// Use only when SSR is not available.
// Prefer static <title> in HTML whenever possible.
function setPageTitle(primary, brand) {
  if (brand === undefined) brand = 'SEO Reference';
  document.title = primary + ' \u2014 ' + brand;
}
setPageTitle('Title Tags');

Anti-patterns

Avoid Correct
Home | My Website Technical SEO Reference for Developers - SEO Reference
SEO Reference | Title Tags | HTML Titles | Title Tag SEO Title Tags - SEO Reference
SEO Reference - Title Tags (brand first) Title Tags - SEO Reference (keyword first)
Identical titles across all pages Unique, descriptive title per page reflecting its primary topic
Title exceeding 70+ characters that will be cut mid-word in SERPs Title under 60 characters that displays in full on both desktop and mobile
