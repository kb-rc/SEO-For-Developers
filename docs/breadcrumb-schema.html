Breadcrumb Schema

BreadcrumbList schema enables breadcrumb trails in SERP results, replacing the URL with a navigational path. This improves CTR and communicates site hierarchy to both users and search engines.

BreadcrumbList JSON-LD

The following schema represents the breadcrumb path for a documentation page. Every docs page on this site includes a BreadcrumbList in the <head> with two items: Home and the current page.

docs/title-tags.html - BreadcrumbList
JSON-LD 
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Home",
      "item": "https://kb-rc.github.io/SEO-For-Developers/"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Title Tags",
      "item": "https://kb-rc.github.io/SEO-For-Developers/docs/title-tags.html"
    }
  ]
}
</script>

Three-level breadcrumb example

For deeper site architectures (category pages, subcategories), include one ListItem per level. Each item must have a position integer that increments by 1, a name, and an item URL for all items except optionally the last.

Three-level BreadcrumbList
JSON-LD 
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Home",
      "item": "https://example.com/"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Blog",
      "item": "https://example.com/blog/"
    },
    {
      "@type": "ListItem",
      "position": 3,
      "name": "Canonical Tags Explained"
    }
  ]
}

HTML breadcrumb markup

The JSON-LD schema must match the visible breadcrumb rendered in the HTML. Google's structured data guidelines require that markup accurately reflects what users can see on the page. Use a semantic <nav> with an ordered list, and optionally add microdata attributes to reinforce the schema inline.

Semantic HTML breadcrumb with microdata
HTML 
<nav aria-label="Breadcrumb">
  <ol
    itemscope
    itemtype="https://schema.org/BreadcrumbList"
  >
    <li
      itemprop="itemListElement"
      itemscope
      itemtype="https://schema.org/ListItem"
    >
      <a
        itemprop="item"
        href="https://kb-rc.github.io/SEO-For-Developers/"
      >
        <span itemprop="name">Home</span>
      </a>
      <meta itemprop="position" content="1" />
    </li>

    <li
      itemprop="itemListElement"
      itemscope
      itemtype="https://schema.org/ListItem"
      aria-current="page"
    >
      <span itemprop="name">Breadcrumb Schema</span>
      <meta itemprop="position" content="2" />
    </li>
  </ol>
</nav>

How breadcrumbs appear in SERPs

When Google validates the BreadcrumbList schema, the SERP result displays the breadcrumb path instead of the raw URL beneath the title. For example, a result for this page would show:

kb-rc.github.io › SEO-For-Developers › docs › Breadcrumb Schema

This gives users immediate context about the page's location in the site hierarchy before they click, which typically improves click-through rates for navigational and informational queries.
