Internal Linking Structure

Internal links distribute PageRank (link equity) across your site, signal topical relevance through anchor text, and determine which pages Googlebot discovers and how frequently it recrawls them. A well-structured internal linking architecture is one of the highest-leverage technical SEO investments.

Principles

  • Use descriptive anchor text. Anchor text is a direct keyword relevance signal. Write anchors that describe the linked page's topic: configure robots.txt correctly rather than click here or learn more. Match the target page's primary keyword when natural.
  • Link from high-authority pages to important pages. PageRank flows through internal links. Pages with many inbound links (your homepage, popular posts) should link to pages you want to rank, not just to the next item in a sequence.
  • Maintain a shallow site architecture. Every page should be reachable within three clicks from the homepage. Deep pages - five or more clicks from the root - receive less crawl budget and rank less consistently.
  • Avoid orphan pages. A page with no internal links pointing to it cannot be discovered by Googlebot via crawling (only via sitemap). Add contextual links to new content from related existing pages.
  • Do not over-link. Excessive internal links per page dilute the PageRank value of each link and create a poor reading experience. Use contextual links that provide genuine navigational value to the reader.

Correct anchor text patterns

Internal link anchor text examples
HTML 
<!-- Good: descriptive anchor text matching target page topic -->
<p>
  Before deploying, ensure you have
  <a href="robots-txt.html">configured robots.txt correctly</a>
  to avoid accidentally blocking Googlebot.
</p>

<!-- Good: keyword-rich anchor matching target h1 -->
<p>
  See the guide on
  <a href="canonical-tags.html">canonical tags and duplicate content</a>
  for handling parameter-based duplicate URLs.
</p>

<!-- Bad: generic anchor with no topical signal -->
<p>
  To learn about robots.txt,
  <a href="robots-txt.html">click here</a>.
</p>

<!-- Bad: anchor matches brand, not topic -->
<p>
  Visit
  <a href="canonical-tags.html">SEO Reference</a>
  for more information.
</p>

rel="nofollow" and rel="ugc"

The rel attribute on links communicates editorial intent to Google. Since September 2019, these values are treated as hints - Google may choose to follow or ignore them based on its own assessment. However, using them correctly remains best practice for editorial transparency.

rel attribute examples
HTML 
<!-- rel="sponsored": paid/affiliate links, clearly commercial -->
<a
  href="https://example.com/product"
  rel="sponsored noopener"
  target="_blank"
>
  Recommended SEO audit tool (affiliate)
</a>

<!-- rel="ugc": user-submitted links in comments or forum posts -->
<a
  href="https://user-submitted-site.com"
  rel="ugc noopener"
  target="_blank"
>
  User profile link
</a>

<!-- Editorial external link: no rel needed if you vouch for it -->
<!-- But add noopener for security when using target="_blank" -->
<a
  href="https://developers.google.com/search/docs"
  rel="noopener"
  target="_blank"
>
  Google Search Central documentation
</a>
